# vue_ecommerce1

- 電子商務網站實作練習專案，
- 本站結帳是模擬並無實際效果，
- 客戶頁面

- 訂單編號搜尋
- 庫存功能

分類彈性

購物結帳流程- 結帳後獲得優惠代碼供下次購物使用

商品管理頁面新增、刪除、修改
訂單管理頁面 修改付款狀態 訂購人資訊

優惠卷管理頁面新增、刪除、修改、新增時隨機產生優惠代碼(也可使用自訂優惠代碼)
